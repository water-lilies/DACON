### Code

```python
model = CatBoostRegressor(iterations=10000,             # 반복횟수
                           learning_rate=0.05,          # 러닝레이트
                           l2_leaf_reg=3.5,             # L2 정규화 계수
                           depth=9,                     # 트리 깊이
                           rsm=0.98,                    # Random subspace method.
                           metric_period=1000,
                           loss_function= 'RMSE',      
                           eval_metric= 'RMSE',         # 성능 측정
                           use_best_model=True,
                           random_seed=42               # 랜덤시드 고정
                          ) 
```
